This folder is for prediction of Binding energy, ORR and OER activity for DAC using GNN:
paper published in Advanced functional materials:  https://doi.org/10.1002/adfm.202203439

Data folder: original data and cleaned data
Models folder: trained model
Results folder: predicted results

1data-clean.ipynb: clean data 
2DAC-OER.ipynb: build model and train model for OER prediction
3DAC-ORR.ipynb: build model and train model for ORR prediction
4DAC-Bindingenergy.ipynb: build model and train model for Binding energy prediction
